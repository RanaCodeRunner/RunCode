Write a Java program to create an interface Encryptable with methods encrypt (String data) and decrypt (String encryptedData) that define encryption and decryption operations. Create two classes AES and RSA that implement the Encryptable interface and provide their own encryption and decryption algorithms.

Code:-



            // Encryptable interface
            interface Encryptable {
                String encrypt(String data);
                String decrypt(String encryptedData);
            }
            
            // AES class implementing Encryptable interface
            class AES implements Encryptable {
                @Override
                public String encrypt(String data) {
                    // Add AES encryption logic here
                    System.out.println("AES Encryption: " + data);
                    return "AES-Encrypted-Data"; // Placeholder for actual encrypted data
                }
            
                @Override
                public String decrypt(String encryptedData) {
                    // Add AES decryption logic here
                    System.out.println("AES Decryption: " + encryptedData);
                    return "Decrypted-Data"; // Placeholder for actual decrypted data
                }
            }
            
            // RSA class implementing Encryptable interface
            class RSA implements Encryptable {
                @Override
                public String encrypt(String data) {
                    // Add RSA encryption logic here
                    System.out.println("RSA Encryption: " + data);
                    return "RSA-Encrypted-Data"; // Placeholder for actual encrypted data
                }
            
                @Override
                public String decrypt(String encryptedData) {
                    // Add RSA decryption logic here
                    System.out.println("RSA Decryption: " + encryptedData);
                    return "Decrypted-Data"; // Placeholder for actual decrypted data
                }
            }
            
            // Main class to demonstrate the usage
            public class EncryptionExample {
                public static void main(String[] args) {
                    // Creating objects of AES and RSA
                    Encryptable aesEncryptor = new AES();
                    Encryptable rsaEncryptor = new RSA();
            
                    // Encrypting and Decrypting using AES
                    String dataToEncrypt = "SensitiveData";
                    String aesEncryptedData = aesEncryptor.encrypt(dataToEncrypt);
                    String aesDecryptedData = aesEncryptor.decrypt(aesEncryptedData);
            
                    // Encrypting and Decrypting using RSA
                    String rsaEncryptedData = rsaEncryptor.encrypt(dataToEncrypt);
                    String rsaDecryptedData = rsaEncryptor.decrypt(rsaEncryptedData);
                }
            }
