Write a Java program to sum values of an array

Code:-



            class a
            {
                static int arr[] = { 12, 3, 4, 15 };
                  
                static int sum()
                {
                    int sum = 0; 
                    int i;
                   System.out.println("array:");
                    for (int j : arr) 
                    {
                       System.out.print(j+"  "); 
                    }
                    
                    for (i = 0; i < arr.length; i++)
                        sum += arr[i];
             
                    return sum;
                }
             
                
                public static void main(String[] args)
                {
                    System.out.println("Sum of given array is "
                                       + sum());
                }
            }
