How to convert Decimal to Hex

Code:-



            public class DecimalToHexExample {
                public static void main(String[] args) {
                    // Example: Convert Decimal to Hexadecimal
                    int decimalNumber = 42; // Replace with your decimal number
            
                    // Using Integer.toHexString() method
                    String hexString = Integer.toHexString(decimalNumber);
            
                    System.out.println("Decimal Number: " + decimalNumber);
                    System.out.println("Hexadecimal Equivalent: " + hexString.toUpperCase());
                }
            }
