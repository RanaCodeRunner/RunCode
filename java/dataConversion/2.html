How to convert int to String

Code:-



            class a {
                public static void main(String[] args) {
                    // Example: Convert int to String
                    int number = 42;
            
                    // Method 1: Using Integer.toString()
                    String strNumber1 = Integer.toString(number);
                    System.out.println("Method 1: Using Integer.toString() - Result: " + strNumber1);
            
                    // Method 2: Using concatenation with an empty string
                    String strNumber2 = "" + number;
                    System.out.println("Method 2: Using concatenation - Result: " + strNumber2);
                }
            }
