Write a Java program to show an example of nested try block.

Code:-



            public class NestedTryExample {
              public static void main(String[] args) {
                  try {
                      // Outer try block
                      System.out.println("Outer try block starts.");
          
                      // Some code that may cause an exception
                      int result = divideNumbers(10, 0);
          
                      // This line will not be executed if an exception occurs above
                      System.out.println("Result: " + result);
          
                      // Inner try block
                      try {
                          System.out.println("Inner try block starts.");
          
                          // Some code that may cause another exception
                          int[] array = new int[5];
                          int value = array[10];
          
                          // This line will not be executed if an exception occurs above
                          System.out.println("Value: " + value);
          
                          System.out.println("Inner try block ends.");
                      } catch (ArrayIndexOutOfBoundsException innerException) {
                          System.err.println("Inner Exception: " + innerException.getMessage());
                      }
          
                      System.out.println("Outer try block ends.");
                  } catch (ArithmeticException outerException) {
                      System.err.println("Outer Exception: " + outerException.getMessage());
                  }
              }
          
              // A method that may throw an ArithmeticException
              private static int divideNumbers(int numerator, int denominator) {
                  return numerator / denominator;
              }
          }
