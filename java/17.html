Write a Java program to print the even numbers from 1 to 20

Code:-



            class p17 {
                public static void main(String[] args){
                for (int i = 1; i < 20; i++) {
                        if (i % 2 == 0)
                        {
                            System.out.println(i);
                        }
                    }
                }
