Write a java program to calculate the sum of first 10 natural number using loop.

Code:-



            public class p107 {
                public static void main(String[] args) 
                {
                    int n = 10;
                    int sum = 0;
            
                    for (int i = 1; i <= n; i++) {
                        sum += i;
                    }
            
                    System.out.println("Sum of the first " + n + " natural numbers is: " + sum);
                }
            }
