Write a Java program to print out all Armstrong numbers between 1 to 600 using loop

Code:-



            class a
            {
                public static void main(String[] arg)
                {
                int i=1,a,arm,n,temp;
                System.out.println("Armstrong numbers between 1 to 600 are");
                while(i<600)
                {
                n=i;
                arm=0;
                while(n>0)
                {
                    a=n%10;
                    arm=arm+(a*a*a);
                    n=n/10;
                }
                if(arm==i)
                    System.out.println(i);
                i++;
                }
                }
            }
