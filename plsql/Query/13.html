Procedure to check the AGE of the Roll_no=1 . If AGE<18 then print "NOT ELIGIBLE FOR VOTING" and if AGE>=18 then print "ELIGIBLE FOR VOTING.

~Create table of students from which we can check if they are eligible to give VOTE or not.

code:-

declare
x number;
begin
select age into x from student where roll=1;
if x>=18 then
dbms_output.put_line('Eligible to give vote');
else
dbms_output.put_line('Noy Eligible to give vote');
end if;
end;
